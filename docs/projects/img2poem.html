Image to Poem

Personal Project

#deeplearning #nlp #python #image #poem

After reading a research paper for poem generation from images I decided to implement it using PyTorch, my go-to framework.

This project was very challenging as it requires knowledge in Natural Language Processing (NLP) and Image Analysis.

In addition, it is a bit thorny to project an image to a Poetic Space. Nevertheless this was resolved in the original paper by using a Generative Adversarial Network with a custom discriminator, classifying real poems from fakes but also classifying images relevant to the generated poem (i.e. a poetic metric).

When : 09.2020
Where : Paris, France
Duration : 3 weeks

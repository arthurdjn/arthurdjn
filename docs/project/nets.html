Neural Networks from Scratch

School Project

#deeplearning #cnn #rnn #dnn #python #numpy #autograd

During my master studies at the University of Oslo I created a Deep Learning Framework from scratch, using only Numpy.

Even though this project started as a school project for IN5400 lecture, known as Machine Learning for Image Analysis.

I wanted to push the project even further and create a complete framework, with full forward and backward propagation for custom networks. I also wanted to create an autograd system, to understand in details the mechanisms used in popular DL frameworks.

When : 03.2020
Where : Oslo, Norway
Duration : 4 weeks

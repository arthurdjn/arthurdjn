PySnake

Personal Project

#deeplearning #python #game #snake #neuralnetwork

PySnake is a python version of the classic snake game. The additional feature, not the least, is its artificial intelligence ability to play autonomously the game. The neural network behinds the snake brain helps to make the best decision from its surrounding.

This project has been the opportunity to develop from scratch a simple neural network, and visualize it. I wanted to make a versatile project, that can be easily reused by others, so I choosed to organize my code into classes and make it entirely OOP.

When : 02.2020
Where : Oslo, Norway
Duration : 40 hours

Context

The game is simple. The player / AI control a snake in four directions : UP, RIGHT, DOWN, LEFT. The goal is to eat as many apples as possible, increasing the snake's length. The snake dies when it eats itself or crash in a wall.

The goal is to make an artificial intelligence that can beat a human player.

A snake in your browser

Javascript

PySnake is a python package, but I remade it entirely in javascript to display it in the browser, so anyone can see the final results.

Play

There are a lot of add-ons, so feel free to test all of them. For a better experience, it is recommended to play on a laptop.

To play the game on your own, just press the New Game button, and play with the directionnal keys. You can visualize the snake's vision by pressing the V key, and modify it with the slider - it will automatically update the snake's brain. In addition, you can let the snake play with Run AI button. You can finally Train the snakes with a custom vision, but it's just for fun. As it is running on your browser, the computation is really slow so the javascript implementation reaches its limits.

Code

The code is commented and available on my public repository, on github. The implementation is entirely OOP like, as it is easier to maintain and update. For example thanks to the last update, you can let a snake evolve within a changing environment.

### Downloader La classe ``Downloader`` est utilisée pour définir une architecture commune entre les différentes APIs.
### Overpass Pour assurer un transfert rapide de données, l'API Overpass a été utilisée.
### Sentinel Les images satellitaires sont récupérées via l'API de Sentinel Hub. La **résolution** varie entre **10m et 60m**.
### MapBox L'API MapBox est également utilisée afin de récupérer des tuiles. La **résolution** dépend des zones géographiques, mais peut aller **jusqu'à 50cm**.
